# Aidoku
A free and open source manga reading application for iOS and iPadOS.

## Features
- [x] Ad free
- [x] Robust WASM source system
- [x] Online reading through external sources
- [x] iCloud sync support
- [x] Downloads
- [x] Tracker support
- [ ] Update notifications
